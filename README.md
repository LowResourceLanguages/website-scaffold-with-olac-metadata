# website-scaffold-with-olac-metadata
php tool to create simple pages using OLAC/Dublincore metadata

This small tool was created to work with a Wordpress and OLAC/Dublincore compliant metadata to make a small form for archive posts.

See this: https://depts.washington.edu/sahteach/postbuilder.php

Here is a paper describing the process:

[**Constructing Online Language Learning Content Archives for
Under-Resourced Language Communities** by Russell Hugo](http://depts.washington.edu/uwwpl/vol33/1-Hugo-Archiving.pdf)
